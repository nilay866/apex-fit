# Apex AI Feature Summary (Old + New)

This file consolidates all documented and implemented features into a single non-code list.

**Core Features (Original / Documented)**
- Authentication: email sign-up, sign-in, and sign-out.
- Profile setup: name, height, weight, and fitness goals.
- Workout tracking: create routines, log sets/reps/weight, and review workout history.
- Nutrition tracking: log meals, calories, and water intake with daily goals.
- AI coach: daily workout suggestions plus feedback and motivation.
- Progress photos: upload and store transformation photos.
- Reports and stats: charts for workouts, nutrition, weight, and measurements.
- Home dashboard: daily progress summary, streaks, and AI suggestions.

**Expanded / Newer Features (Implemented in Codebase)**
- Social feed with activity and friend interactions.
- Cardio and run tracking with live maps.
- Apple Health integration.
- Exercise library with detailed exercise pages.
- Exercise animations and visual guidance.
- Circuit player for guided interval/circuit sessions.
- Routine library and routine editor.
- Workout programs and structured plans.
- Achievements and streak tracking.
- Adaptive logic for personalization and recommendations.
- Local exercise catalog for faster/offline browsing.
- Exercise form guide content.
- Multi-provider AI support (Gemini and AWS Bedrock).

**Platform & Data Capabilities**
- Supabase-backed auth, profiles, and workout data sync.
- Cloud storage for progress photos and user data.
- Reports built from stored workout and nutrition history.
