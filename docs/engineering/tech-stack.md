# Tech Stack

This document lists the key technologies, libraries, and frameworks used in the Apex AI project.

## Frontend

*   **[Flutter](https://flutter.dev/):** The UI framework used to build the mobile application for both iOS and Android.
*   **[Dart](https://dart.dev/):** The programming language used to develop the Flutter application.

## Backend

*   **[Supabase](https://supabase.com/):** The Backend-as-a-Service (BaaS) platform used for the backend. Supabase provides:
    *   **PostgreSQL Database:** For data storage.
    *   **Authentication:** For user management.
    *   **APIs:** Auto-generated REST and GraphQL APIs.

## Key Libraries and Packages

*   **[supabase_flutter](https://pub.dev/packages/supabase_flutter):** The official Flutter library for Supabase.
*   **[google_fonts](https://pub.dev/packages/google_fonts):** For using custom fonts from Google Fonts.
*   **[integration_test](https://pub.dev/packages/integration_test):** For writing integration tests.
*   **[flutter_test](https://api.flutter.dev/flutter/flutter_test/flutter_test-library.html):** For writing unit and widget tests.
*   **[http](https://pub.dev/packages/http):** For making HTTP requests to the AI service.

This is not an exhaustive list of all the packages used in the project. For a complete list, please refer to the `pubspec.yaml` file.
